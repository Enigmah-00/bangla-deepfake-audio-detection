# KonthoProhori Architecture

This folder contains the professional architecture package for the SciBlitz AI
Challenge submission. It is intended for judges, reviewers, deployers, and team
members who need to understand how the product works beyond the user interface.

## Files

- `SYSTEM_ARCHITECTURE.md` - product-level system design and module boundaries.
- `INFERENCE_PIPELINE.md` - exact audio preprocessing and 227-feature model path.
- `API_CONTRACT.md` - public FastAPI endpoint contract for frontend integration.
- `DEPLOYMENT_ARCHITECTURE.md` - Railway backend and Vercel frontend deployment.
- `SECURITY_PRIVACY.md` - privacy, safety, and responsible-use architecture.
- `diagrams/inference-flow.mmd` - Mermaid source for the inference flow.
- `diagrams/deployment-flow.mmd` - Mermaid source for deployment topology.

The generated PNG diagrams used in the README/report live in `assets/`.
